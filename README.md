This repository contains a Windows Forms (GUI) application built in C# using Visual Studio, designed to educate organizational users on digital safety. A customized user interface styled with a distinctive pink, white, and black color palette, optimized for high scannability and visual appeal. Features a structured backend knowledge engine that dynamically accepts user registration, parses natural input keywords, and provides targeted, precise responses regarding Phishing, Firewalls, and Malware. Enhances user onboarding by executing a verbal greeting (Buhle.wav) on application initialization. Implements a strict separation of concerns between visual layout presentation logic (MainForm.cs) and core dictionary evaluation logic (ChatBot.cs).
